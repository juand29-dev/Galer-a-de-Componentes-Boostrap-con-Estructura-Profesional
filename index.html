Accordion component


Este es el cuerpo del primer elemento del acordeón. Se muestra por defecto hasta que el plugin de colapso añade las clases correspondientes que usamos para dar estilo a cada elemento. Estas clases controlan la apariencia general, así como la muestra y el ocultamiento mediante transiciones CSS. Puedes modificar cualquiera de esto con CSS personalizado o sobrescribiendo nuestras variables por defecto. También vale la pena notar que casi cualquier HTML puede ir dentro de .accordion-body, aunque la transición limita el overflow.

Alert component


Muestra mensajes informativos, de éxito, advertencia, error o neutros. Pueden ser dismissible (cerrables). y sirve para la Retroalimentación inmediata tras acciones: formularios enviados, errores, confirmaciones.

Notificaciones


Pequeña etiqueta o contador que suele adjuntarse a botones o títulos. y sirve para mostrar cantidades (notificaciones), estados (nuevo, beta) o resaltar etiquetas.





Migaja de pan
